<h2>프론트엔드 (Front-end)</h2>
<ul>
  <li>Jsp : Spring Mvc패턴의 view단으로 사용하기위해 템플릿 역할로 사용하였습니다.</li>
  <li>Javascript(vue.js, ajax) : 사원메일조회, 게시판(게시글, 댓글)/캘린더CRUD의 비동기처리를 하기위해 사용하였습니다.</li>
  <li>CSS : 웹 페이지의 스타일과 레이아웃을 정하기 위해 사용하였습니다.</li>
